# gerenciador-gastos

Projeto desenvolvido para praticar JavaScript.

## Conceitos utilizados

- Variáveis
- Constantes
- Arrays
- Strings
- Operadores aritméticos
- Métodos matemáticos
- Data e hora

## Funcionalidades

- Cálculo do total de gastos
- Identificação do maior gasto
- Exibição de data e hora
- Relatório final no console

## Tecnologias

- JavaScript
